<h2>Introduction</h2>

<p>Git is a great tool. Each programmer has to know the concept of version control system. Git (or other <abbr title="distributed version control system">DVCS</abbr>) ensure:</p>

<ul>
    <li>The place where we can keep our code</li>
    <li>The backup copy</li>
    <li>More efficient work in the team</li>
    <li>Insight into the whole project history</li>
</ul>

<p>As you can see there are a lot of benefits of using git, but in order to use the potential of git you need to follow a few rules whcih I will describe below.</p>
